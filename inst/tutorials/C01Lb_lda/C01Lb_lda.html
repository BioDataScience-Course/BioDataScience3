Ce tableau comprend 4 variables. La variable gender est une variable facteur avec 2 niveaux : M (men), W (Women). Il y a également 3 variables numérique qui sont les attributs sur les items :

  • weight : la masse en kg
  • height : la taille en cm
  • wrist : la circonférence du poignet en mm

Cet tableau est divisé en un set d’apprentissage et un set de test. Le set d’apprentissage comprend 2/3 du tableau principal. Ce tableau nommé bio_learn comprend 262
